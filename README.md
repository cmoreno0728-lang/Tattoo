# Atelier Ink — GitHub Pages Site

Minimal static website for a premium tattoo studio concept.

## Pages
- `index.html` — Home
- `artists.html` — Artist profiles
- `contact.html` — Contact details

## Deploy on GitHub Pages
1. Push this repository to GitHub.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Select your default branch (e.g., `main`) and folder `/ (root)`.
5. Save. Your site will publish at:
   - `https://<username>.github.io/<repo-name>/`

## Customize
- Replace placeholder contact details in `contact.html`.
- Replace artist names and bios in `artists.html`.
- Update Open Graph URL in `index.html` once your repo URL is known.
